Adobe Firefly & IA dans la suite Creative Cloud

⏱ 80 min Adobe Firefly Photoshop Illustrator Express
CRCN-Édu travaillées
2.2Concevoir ressources 3.1Concevoir situations 4.3Engager apprenants
🎯
Objectif : Intégrer Adobe Firefly dans un workflow Photoshop et Illustrator professionnel. Maîtriser le Generative Fill, la vectorisation IA et Adobe Express pour produire des assets pédagogiques et créatifs rapidement.

🖼️ Activité 1 — Generative Fill dans Photoshop (25 min)
1
Ouvrez une photo produit ou une image de signalétique dans Photoshop (CC 2024 minimum requis). Sélectionnez une zone avec l'outil de sélection rectangulaire ou lasso.
2
Generative Fill : Dans la barre contextuelle, cliquez sur "Remplissage génératif". Décrivez ce que vous voulez (ex : "urban street background at golden hour"). Firefly génère 3 variantes — choisissez la meilleure.
3
Generative Expand : Avec l'outil Recadrage, étendez la toile au-delà des bords de l'image. Sélectionnez les zones vides → Remplissage génératif sans prompt → Firefly prolonge la scène de façon cohérente.
4
Remove Object : Sélectionnez un élément indésirable (câble, personne, logo). Touche Suppr → "Remplissage génératif" vide → Firefly reconstruit le fond. Comparez avec l'outil Tampon classique.
5
Cas pédagogique : Créez un visuel pour une affiche signalétique de votre établissement. Partez d'une photo du couloir, étendez-la, ajoutez des éléments générés. Exportez en JPEG haute résolution.
Prompts Firefly — Photoshop Generative Fill
// Fond pour signalétique modern school corridor with natural light, clean minimalist, no people, architectural photography // Extension d'affiche événementielle continuation of a vibrant graphic poster background, bold colors, geometric patterns, French school aesthetic // Remplacement de ciel dramatic sunset sky with warm orange and pink clouds, high contrast, cinematic // Ambiance vitrine commerce elegant retail window display background, soft lighting, luxury feel, neutral tones // Décors pour communication visuelle abstract colorful background suitable for typography overlay, gradient mesh, contemporary design
✏️ Activité 2 — Illustrator : Text to Vector & Recolor (25 min)
1
Text to Vector (bêta) : Dans Illustrator, ouvrez le panneau "Firefly" (Fenêtre → Firefly). Tapez : "pictogramme d'une école avec une plume et une étoile, style ligne épuré, monochrome". Générez des icônes vectorielles directement dans Illustrator.
2
Generative Recolor : Ouvrez un logo ou une illustration vectorielle existante. Sélectionnez-la → Edition → Recoloriage → "Recoloriage génératif". Décrivez : "couleurs d'automne chaleureuses" ou "palette pastel scandinave".
3
Déclinaison de charte : Prenez votre logo (ou celui de l'établissement). Générez 5 variantes colorées pour différents supports (web, print, fond sombre). Exportez chaque variante en SVG et PNG.
4
Retype : Dans Illustrator, sélectionnez du texte vectorisé (converti en tracé). Utilisez Retype pour identifier la police d'origine et proposer des alternatives similaires. Idéal pour les logos clients sans fichier source.
⚡ Activité 3 — Adobe Express pour la communication rapide (15 min)
1
Compte éducation : Si vous avez Adobe CC, Express est inclus. Connectez-vous sur express.adobe.com. Créez un post Instagram pour un événement de votre établissement via le bouton "+ Créer".
2
Magic Write : Dans Express, cliquez sur l'icône texte → "Magic Write". Décrivez : "Texte accrocheur pour une journée portes ouvertes d'une école de design, ton dynamique, 2 lignes max".
3
Remove Background : Importez une photo de produit ou de personne. Cliquez "Supprimer l'arrière-plan" en un clic. Placez le sujet sur un fond généré par Firefly. Exportez pour les réseaux.
4
Redimensionnement intelligent : Créez votre visuel une fois (format Instagram carré). Utilisez "Redimensionner" → sélectionnez Story, LinkedIn, et affiche A4. Express adapte automatiquement la composition.
💡 Conseils formateur — Firefly en classe
Firefly est entraîné uniquement sur des contenus licenciés Adobe Stock — c'est l'argument légal clé face aux élèves qui utilisent d'autres outils.
Les contenus générés sont "Firefly générés" — les élèves doivent le mentionner dans leurs dossiers de certification.
Pour les CAP et Bac Pro, Express est le point d'entrée idéal : pas de courbe d'apprentissage, résultats immédiats.
Illustrator Text to Vector est encore en bêta : les résultats varient selon la description. Insistez sur l'itération et le prompt engineering.
Firefly respecte les styles des artistes vivants — contrairement à Midjourney, on ne peut pas demander "dans le style de [artiste vivant]".

🎨 Activité 1 — Midjourney : moodboard & direction artistique (25 min)
1
Accès : Rendez-vous sur midjourney.com → "Start creating" (abonnement ou essai). Utilisez l'interface web directement (plus besoin de Discord depuis 2024).
2
Premier moodboard : Entrez le prompt ci-dessous pour une campagne fictive. Générez 4 images. Sélectionnez la meilleure → cliquez U1-U4 pour upscaler, V1-V4 pour des variantes.
3
Paramètres avancés : Testez --ar 16:9 (format paysage), --style raw (moins stylisé), --chaos 30 (plus de diversité). Comparez les résultats.
4
Style reference : Glissez une image existante dans Midjourney pour l'utiliser comme référence de style (--sref URL). Créez une série cohérente pour une marque fictive.
5
Export & usage : Téléchargez vos images. Importez-les dans Photoshop pour finalisation. Discutez des droits : Midjourney Basic = usage commercial autorisé.

🔬 Activité 2 — Leonardo.ai & cohérence de personnage (20 min)
1
Compte gratuit : Créez un compte sur leonardo.ai (150 tokens/jour gratuits). Explorez les modèles disponibles : Flux, Leonardo Phoenix, Kino XL selon le rendu souhaité.
2
Elements (cohérence) : Dans "Image Guidance", utilisez "Character Reference" pour maintenir un personnage cohérent entre plusieurs images. Idéal pour les projets BD ou communication avec mascotte.
3
Canvas : Ouvrez le Canvas Leonardo. Générez une illustration, puis avec Inpainting, modifiez seulement une zone (remplacer un vêtement, changer le fond) sans toucher au reste.
4
Projet synthèse : Créez une série de 4 visuels cohérents pour une marque fictive (même style, même univers). Assemblez-les dans un PDF de présentation sous InDesign ou Canva.
✨ Activité 3 — Gemini : améliorer une carte avec l'IA (20 min)
📎
Activité avec pièce jointe : Photographiez une carte réalisée à la main par un élève sur papier quadrillé (croquis de géographie, carte de synthèse…). Uploadez cette image dans Gemini pour obtenir une version retravaillée et mise en page proprement par l'IA — figurés colorés, légende en cursive, symboles reproduits fidèlement.
Exemple : carte de synthèse sur l'Amazonie réalisée par un élève
1
Photographier la carte : Prenez en photo une carte d'élève sur papier quadrillé (exemple : carte de synthèse sur l'Amazonie avec forêts, métropoles, axes de communication). Cadrez bien, lumière naturelle de préférence.
2
Ouvrir Gemini : Rendez-vous sur gemini.google.com. Cliquez sur l'icône 📎 (pièce jointe) pour uploader votre photo. Attendez le chargement complet de l'image.
3
Saisir le prompt : Collez le prompt ci-dessous (bouton Copier) et envoyez. Gemini va générer une version retravaillée : figurés colorés, légende en belle écriture cursive, symboles proprement reproduits.
4
Analyser et discuter : Comparez la carte originale et la version IA. Que l'IA a-t-elle amélioré ? Qu'a-t-elle mal interprété ? Quels usages pédagogiques imaginez-vous en géographie, histoire-géo ou SVT ?
Prompt Gemini — Carte géographie (pièce jointe requise)
Tu es professeur de géographie. Tu t'adresses à des élèves de seconde. Soit donc le plus clair possible et reste simple. Tu vas réaliser une image d'une carte sur papier quadrillé. Tu utiliseras les figurés suivants : forêt, métropoles, axes de communication. La légende sera bien écrite à la main (jolie écriture cursive). Tu garderas tout ce qui est écrit et les symboles et figurés que tu colorieras ou reproduiras proprement.
⚖️
Droits d'auteur & IA — point essentiel à enseigner : En France, les images générées par IA ne sont pas protégeables par le droit d'auteur (l'auteur doit être humain). Les élèves peuvent utiliser ces images commercialement selon les CGU de chaque outil, mais doivent mentionner l'usage de l'IA dans leurs travaux de certification. Midjourney Basic = commercial OK. Firefly = commercial OK (Adobe indemnise). Stable Diffusion modèles publics = vérifiez la licence du modèle.

Objectif : Créer une stratégie de contenus réseaux sociaux assistée par IA, générer des visuels et des textes cohérents, planifier et analyser les performances. Applicable directement aux projets BTS Communication et Bachelor.
🎨 Activité 1 — Canva Magic Studio (20 min)
1
Magic Design : Sur canva.com, cliquez "+ Créer un design" → tapez votre besoin : "Post Instagram pour une exposition de travaux d'étudiants en design". Magic Design génère automatiquement 8 propositions de mise en page avec visuel et texte.
2
Magic Write : Dans un design, cliquez sur une zone de texte → "Magic Write" (icône étoile). Prompt : "Légende Instagram engageante pour une affiche de journée portes ouvertes, 150 caractères, 5 hashtags pertinents".
3
Magic Resize : Créez votre post Instagram (1080×1080). Sélectionnez-le → "Redimensionner et Magic Switch". Choisissez : Story (9:16), LinkedIn (1200×628), et X/Twitter. Canva adapte la composition automatiquement.
4
Brand Kit IA : Si vous avez Canva Pro, importez un logo et une palette couleur dans "Charte graphique". Canva adapte automatiquement tous les templates à vos couleurs. Testez avec la charte de l'établissement.
📊 Activité 2 — Predis.ai : générer un mois de contenu (20 min)
🤖
Predis.ai génère en une seule action : le visuel, le texte, les hashtags et le planning d'une semaine complète de posts. C'est l'outil de référence pour les BTS Communication qui gèrent les réseaux d'une organisation.
1
1
Créer un compte : predis.ai → compte gratuit (10 posts/mois). Créez un "Brand" : nom, secteur (Education/Design), charte couleur, ton de communication.
2
Générer une série : "Créer du contenu" → choisissez "Post unique" → décrivez : "Présenter les formations design et communication de notre école aux futurs lycéens, ton inspirant, visuels modernes". Predis génère visuels + textes + hashtags.
3
Calendrier éditorial : Dans "Planifier", demandez à l'IA de générer un plan de contenu sur 4 semaines pour une campagne de communication d'établissement (Portes Ouvertes, concours, résultats étudiants…).
4
Analyse concurrents : Dans "Analyser", entrez le compte Instagram d'un établissement similaire. Predis analyse le type de contenus performants, les horaires optimaux, les hashtags utilisés.
Prompts réseaux sociaux — BTS Communication
// Stratégie éditoriale (Claude ou ChatGPT) Je gère les réseaux sociaux d'un lycée de design et communication à Aurillac (Cantal, 500 élèves, formations CAP à Mastère). Crée un plan éditorial pour Instagram sur 4 semaines avec : - 3 posts par semaine (lundi, mercredi, vendredi) - Pour chaque post : thème, format (carrousel/reel/image), légende (200 car. max), 8 hashtags - Objectif : attirer des lycéens 15-18 ans et valoriser les travaux d'étudiants - Tons : inspirant, authentique, humain // Rédaction de post engagement Écris 3 versions d'une légende Instagram pour présenter ce projet étudiant : [DESCRIPTION DU PROJET] Format : accroche + 2-3 phrases + CTA + 8 hashtags Ton : enthousiaste mais professionnel Maximum 200 mots par version // Brief créatif pour visuels sociaux Génère un brief créatif pour une série de 5 posts Instagram présentant nos formations. Inclus : direction artistique, palette couleurs, style typographique, cadrage photo, ambiance globale. Cible : 16-22 ans intéressés par les métiers créatifs.
💡 Outils complémentaires réseaux sociaux

Créer des sites & interfaces numériques avec l'IA

CRCN-Édu travaillées

🎯
Objectif : Créer un site web professionnel ou un prototype d'interface en partant d'un prompt texte. Comparer les approches no-code IA (Framer) et design UX (Figma AI) pour des projets DN MADE Numérique et BTS Communication.
🌐 Panorama des outils web IA

Framer AI — site en 30 secondes

Principe : décrivez votre site en texte → Framer génère un site complet, hébergé, responsive, avec animations.

  • Gratuit jusqu'à 1 site publié
  • Idéal : portfolios, sites d'événements, landing pages
  • CMS intégré pour les actualités
  • Export en code React si besoin

Figma AI — design system & prototypage

Principe : l'IA dans Figma aide à créer des composants, auto-layout, et générer des maquettes de wireframe.

  • Plugin Galileo AI : maquette depuis texte
  • Auto Layout intelligent
  • Génération de contenu fictif (noms, avatars, textes)
  • Copilot Figma (bêta 2024) : chat pour modifier
🚀 Activité 1 — Framer AI : créer un portfolio en 20 min
1
Générer le site : framer.com → "Start with AI". Entrez : "Portfolio créatif pour un étudiant en DN MADE Graphisme. Sections : À propos, Projets (grille), Contact. Design minimaliste, typographie bold, fond blanc cassé."
2
Personnaliser : Changez les couleurs dans le panneau "Style" pour correspondre à une charte. Remplacez les images placeholder par des travaux réels (ou générés par Midjourney à l'atelier 2).
3
CMS Projets : Dans Framer, ajoutez une collection "Projets" (CMS). Configurez les champs : titre, description, image, catégorie. Liez la grille de la page Projets à cette collection.
4
Publier : Cliquez "Publish" → obtenez une URL framer.app. Testez sur mobile. Partagez l'URL. Discutez : en quoi ce site est-il un livrable professionnel crédible ?
✏️ Activité 2 — Figma AI : maquetter une app mobile (20 min)
1
Plugin Galileo AI : Dans Figma (compte gratuit), installez le plugin "Galileo AI" (Plugins → Chercher). Lancez-le et décrivez : "Application mobile de suivi des projets étudiants pour une école de design. Écrans : liste des projets, détail projet, profil étudiant."
2
Design tokens : Galileo génère des frames Figma complètes. Explorez le design system créé automatiquement (couleurs, typographie, composants). Identifiez les composants réutilisables.
3
Prototype interactif : Reliez les écrans entre eux avec les connecteurs Figma (cliquez l'icône flèche en haut à droite). Créez un parcours utilisateur de 3 écrans. Partagez le lien de prototype.
🌀 Bonus — Spline AI : interface 3D interactive (10 min)
🔮
Spline (spline.design) permet de créer des scènes 3D interactives embarquables dans n'importe quel site web. Avec l'IA, vous générez des objets 3D en décrivant leur forme et matériau.
1
Créez un compte Spline gratuit. Ouvrez un nouveau projet. Dans la barre gauche, cliquez l'icône IA → "Generate 3D object". Décrivez : "floating geometric crystal, glass material, rotating slowly".
2
Ajoutez une interaction : sélectionnez l'objet → "Events" → "Mouse Hover" → effet "Scale up". Exportez le code d'intégration iframe et collez-le dans votre site Framer.
💡 Pour les DN MADE Numérique — aller plus loin
Webflow + Relume : relume.io génère des wireframes et des sites Webflow complets depuis un prompt. Plus technique que Framer, mais plus de contrôle CSS.
v0.dev (Vercel) : génère des interfaces React/Tailwind depuis une description. Pour les étudiants qui veulent du code propre.
Rive : animations interactives avancées avec AI Assist. Export web natif. Parfait pour les micro-interactions dans les portfolios DN MADE.
Uizard : transformer un croquis en wireframe numérique via photo. Très utile en phase d'idéation rapide.

🎨 Activité 1 — Construire une palette & système typographique (15 min)

Khroma — palette IA personnalisée

khroma.co apprend vos préférences couleur (vous notez 50 couleurs) puis génère des milliers de palettes sur mesure. Exportez en HEX, RGB, ou directement en palette Adobe.

  • Gratuit après calibration
  • Palettes 2, 3 ou 4 couleurs
  • Prévisualisation sur typographie, poster, photo
  • Export direct en Adobe Swatch (.ase)

Fontjoy — association typographique IA

fontjoy.com génère des associations titre/sous-titre/corps de texte harmonieuses. L'IA calcule la "distance visuelle" entre les polices pour créer des contrastes équilibrés.

  • Gratuit, toutes Google Fonts
  • Verrouiller une police, générer des alternatives
  • Régler le contraste souhaité (harmonieux ↔ contrasté)
  • Copier les imports CSS directement
1

⚖️ Éthique & droits — les points essentiels pour les métiers créatifs

❌ Ce qu'on ne peut PAS faire

  • Générer des images "dans le style de [artiste vivant]" sur Firefly (bloqué)
  • Utiliser des images IA dans des concours qui l'interdisent explicitement
  • Prétendre qu'un visuel IA est une photo réelle dans un contexte publicitaire (Loi Sapin II)
  • Vendre des images IA via Midjourney Basic sans mention explicite

⚠️ Zone grise à enseigner

  • Les certifications (CAP, BTS) : les référentiels ne précisent pas encore la place de l'IA — vérifiez les consignes d'examen annuellement
  • Copyright : en France, une image IA n'est pas protégeable — mais la composition finale avec IA peut l'être si l'humain a créé une œuvre originale
  • Midjourney v6 et les styles d'artistes : techniquement possible, éthiquement discutable

✅ Bonnes pratiques professionnelles

  • Mentionner "Créé avec assistance IA" dans les dossiers de certification
  • Utiliser Firefly pour les projets clients (couverture légale Adobe)
  • Garder les prompts et itérations comme preuve du processus créatif
  • Former les élèves à évaluer critiquement les sorties IA (Midjourney hallucine les mains, les textes, les logos)
💡 Pour le Mastère DA — IA comme outil de compétitivité
Un DA qui maîtrise l'IA peut produire des pitchs visuels 5× plus vite : moodboards Midjourney en 10 min vs 2h de sourcing manuel.
L'IA ne remplace pas le jugement esthétique — elle amplifie la capacité de production. La valeur du DA est dans la sélection, pas dans la génération.
Les studios qui recrutent en 2025 demandent "maîtrise des outils IA génératifs" dans 67% des offres (LinkedIn Talent Insights, Jan 2025).
Enseigner l'IA dans les formations créatives, c'est aussi enseigner la pensée critique : reconnaître ce qui est générique, what makes it "too AI-looking", et comment l'humaniser.
